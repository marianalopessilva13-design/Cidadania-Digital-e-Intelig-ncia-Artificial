# ⚡ Cidadania Digital Y2K

## 🧠 Objetivo
Este projeto é um portal educativo interativo com estética retrô anos 2000 (Y2K), criado para conscientizar sobre:

- Deepfakes gerados por IA
- Desinformação digital
- Verificação de fatos
- Impacto das fake news na sociedade

## 🎨 Estilo
Inspirado na web dos anos 2000:
- Cores neon vibrantes
- Layout caótico organizado
- Efeitos glow e glitch
- Interface estilo arcade / early internet

## ⚙️ Tecnologias
- HTML5
- CSS3 (Flexbox + Responsividade)
- Vanilla JavaScript

## 📊 Funcionalidades
- Quiz interativo com correção imediata
- Estatísticas animadas
- Formulário com resposta dinâmica
- Modo escuro (Dark Mode)
- Barras de impacto visual
- Contador animado

## 🎥 Mídia
Inclui embed de vídeo educativo via YouTube.

## 📱 Responsividade
Adaptado para:
- Mobile
- Tablet
- Desktop

## 📌 Créditos
Projeto educacional criado para fins de conscientização digital e exploração estética Y2K.
